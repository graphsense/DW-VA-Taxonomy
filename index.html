This site is a collaborative, community activity with a mission to create, maintain, and promote a standardized taxonomy of semantic concepts used in forensic investigations related to the Dark Web and Cryptocurrencies.

The taxonomies and concepts defined on this page are intended to be used in tools and thus create a degree of semantic interoperability between exchanged data. Therefore, all taxonomies are published in a human readable form (this is what you see right now) and provided as download in various machine readable formats.

As far as possible, the taxonomies build on existing definitions and do not claim to replace them. Rather, they represent an orthogonal approach to try to define and standardize existing terminology that is already used in tools.

For further information on the collaborative taxonomy editing process, please visit the Git repository this site is generated from.
